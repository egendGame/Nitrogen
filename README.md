# Nitrogen
Simple nitro generator coded in python and can create 1000 nitro links in 2sec

![trial](srcs/gif.gif)


## Git Installation
```
# clone the repo
$ git clone https://github.com/thelinuxuser-choice/Nitrogen
# change the working directory to Nitrogen
$ cd Nitrogen

```
## Usage

```
python3 nitrogen.py
```
